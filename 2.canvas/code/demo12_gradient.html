知识点：

注意，渐变创建的只是一个颜色

使用createLinearGradient来创建线性渐变，参数是两个点来缺点渐变的范围

通过addColorStop来给渐变添加颜色...参数第一个是0-1的数，表示这个颜色最深的地方在哪，第二个参数是颜色

使用createRadialGradient来创建径向渐变，参数是两个圆来缺点渐变的范围

通过addColorStop来给渐变添加颜色...参数第一个是0-1的数，表示这个颜色最深的地方在哪，第二个参数是颜色

渐变设置好后不会更改，它只是一个颜色，需要通过其他方式来使用

var lg=ctx.createLinearGradient(0,0,0,300);
lg.addColorStop(0,'red');
lg.addColorStop(0.5,'blue');
lg.addColorStop(1,'green');
